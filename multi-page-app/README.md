# Multi-Page React App

## Overview
This project is a Multi-Page Single Page Application (SPA) developed using React and React Router DOM. The application demonstrates client-side routing, allowing users to navigate between multiple pages without reloading the browser.

The project contains multiple pages such as Home, About, and Contact, which are created as separate React components. Navigation is handled using React Router for a smooth and fast user experience.

---

## Features
- Multi-page navigation using React Router
- Single Page Application (SPA)
- Responsive navigation bar
- Reusable React components
- Fast page transitions without reload
- Simple and clean user interface

## Installation Steps

### Step 1: Create React App

```bash
npx create-react-app multi-page-app
````

### Step 2: Open Project Folder

```bash
cd multi-page-app
```

### Step 3: Install React Router

```bash
npm install react-router-dom
```

### Step 4: Start Development Server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
